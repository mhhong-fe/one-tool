# 每日打卡 · one-tool

个人每日打卡工具，支持多类目、换算分数、热力图统计，数据保存在本地。

## 功能

### 总览
- 今日完成度与问候
- 本周 / 本月打卡、今日得分
- 未打卡类目提醒

### 打卡
- 选择类目，填写详情（数字或文本）
- 数字类目支持自定义换算分数（如：每 3 公里 = 1 分）

### 统计
- **打卡热力图**：按分数深浅展示，色阶对应 0 / 1～3 / 3～6 / 6～10 / 10+ 分，点击格子可查看当日明细
- **年度趋势**：各月打卡柱状图、类目占比（按分数）

### 设置
- 类目管理：新增、编辑、删除
- 换算分数：为数字类目配置「每 N 单位 = 1 分」
- 数据备份与恢复、清空

## 默认换算

| 类目     | 规则        |
|----------|-------------|
| 跑步     | 3 km = 1 分 |
| LeetCode | 1 题 = 1 分 |
| 读书     | 20 页 = 1 分 |

## 技术栈

- Vue 3 + Vite + Vue Router
- Naive UI
- dayjs
- localStorage
